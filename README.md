# QA-Cademy API Automation

## Getting Started
Install the npm dependencies
````
npm install
````
## Suggested Extensions VS code
Open File/Preferences/Extensions
````
Code Runner
Tabine
Cucumber
````
## Run Jest tests
````
npm test
````
## Run Cucumber tests
````
npm run bddTests
````

````
CTR + ALT + N
````
